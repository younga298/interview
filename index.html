project

2025.06.20 ~ 06.24

제작인원 :1명

기여도 100%

SKILL

HTML, CSS, jQuery, Js

project PROCEDURE

메뉴 구조는 CSS와 JS를 적절히 분리해 구현했습니다. hover·focus로 처리 가능한 영역은 CSS로, 포커스가 벗어나도 유지되어야 하는 경우는 JS로 classList.toggle()을 활용해 제어했습니다. 서브페이지의 메뉴 토글 기능은 클릭 이벤트 충돌을 방지했고, 서브메뉴가 있는 항목 클릭 시에만 메뉴가 열리고 닫히도록 설정했고, 내부 항목(서브메뉴 항목)을 클릭했을 때는 메뉴 토글이 되지 않도록 return 처리를 넣어 이벤트를 정교하게 분리했습니다. Swiper는 breakpoints로 반응형 슬라이드 수를 조절하고, resize 이벤트에 swiper.update()를 적용해 실시간 반영되도록 했습니다. 페이지네이션 위치 문제는 Swiper 기본 스타일을 확인해 position: static으로 재정의해 깔끔하게 정렬했습니다. 레이아웃 전반에는 min-height, rem 단위를 활용해 폰트와 여백이 디바이스 해상도에 따라 자연스럽게 조정되도록 했습니다.
